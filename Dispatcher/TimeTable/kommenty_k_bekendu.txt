GET
- без параметров
- возвращает список всех рейсов в типе List<Flight>

GET (string what, string date)
what == ports -> выдает список названий аэропортов типа List<String>
what == planes -> выдает список названий самолетов типа List<String>
what == arrival -> выдает список прибывающих СЕГОДНЯ (date) рейсов типа List<Flight>
what == departure -> выдает список улетающих СЕГОДНЯ (date) рейсов типа List<Flight>

POST 
в параметрах запроса (Request.Content) - запись типа Models.Flight - значит в ней _номера_ типов самолетов и аэропортов, которые нужно брать из комбобокса, сформированного на основе запроса GET (ports, ...), GET (planes, ...)
эта запись просто добавляется в таблицу рейсов
